# 💻 Módulo de Programación - 1º DAM
### Desarrollado por: Adrián Fambuena Palomo

Este repositorio contiene algunos ejercicios, prácticas y retos de código realizados en Java durante el primer curso de Desarrollo de Aplicaciones Multiplataforma.
---

## 🛠️ Tecnologías y Herramientas
* **Lenguaje:** Java (JDK 17+)
* **Entorno:** IntelliJ IDEA
* **Conceptos:** Lógica de programación y POO.

---

## 📁 Contenidos del Repositorio

### 1. Fundamentos y Estructuras
* **Estructuras de control:** Uso de bucles (`for`, `while`), condicionales (`if`, `switch`) y operadores lógicos.
* **Arrays y Matrices:** Manipulación de datos en estructuras simples y multidimensionales.

### 2. Programación Orientada a Objetos (POO)
* **Clases y Objetos:** Diseño de modelos, constructores y métodos.
* **Pilares POO:** Aplicación de Herencia, Polimorfismo, Encapsulamiento y Abstracción.
* **Interfaces:** Definición de contratos y comportamientos modulares.

### 3. Gestión Avanzada
* **Colecciones:** Uso de `ArrayList`, `HashMap` y otras estructuras dinámicas.
* **Excepciones:** Control de errores y robustez del código.
